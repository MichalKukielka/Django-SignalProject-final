What is this app?

Presented web application is a project created on the basis of the Django Framework in version 2.0.3. It uses the PostgreSQL 10 database. It is a solution prepared to transfer complicated calculations related to data analysis to the cloud. On several examples, the application shows that it is possible to perform efficient data analysis in easy way.
If you are a new user, do not hesitate to check helpdesk to obtain crucial information.
